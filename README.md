# Motion Viewer

## 기능

1) 모션을 프레임 단위로 재생할 수 있는 기본적인 UI를 제공한다. 
2) 모션 데이터 중 하나인 Bvh 파일을 읽어 재생시킬 수 있다.
3) 기존 모션에 다양한 캐릭터 애니메이션 기법(warping / stitching / blending)을 적용할 수 있다.
4) 간단한 mass-spring 모델을 시뮬레이션 해볼 수 있다.
